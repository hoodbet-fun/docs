# Wallet & Robinhood Chain

HoodBet runs on **Robinhood Chain mainnet** (chain ID `4663`).

## Network details

| Parameter | Value |
|-----------|-------|
| Chain ID | `4663` |
| RPC | `https://rpc.mainnet.chain.robinhood.com` |
| Explorer | [robinhoodchain.blockscout.com](https://robinhoodchain.blockscout.com) |
| Native gas token | ETH |

## Add network to MetaMask

1. Open MetaMask → **Networks** → **Add network** → **Add a network manually**
2. Fill in the values from the table above
3. Save and switch to Robinhood Chain

## Robinhood Wallet

If you use Robinhood Wallet, Robinhood Chain may already be available. Connect at [app.hoodbet.fun](https://app.hoodbet.fun) and approve the connection.

## Gas

Transactions (deposit, withdraw, claim) require a small amount of **ETH** on Robinhood Chain for gas.

## Security tips

- Never share your seed phrase or private key
- Always verify you're on **app.hoodbet.fun** or **hoodbet.fun**
- Confirm chain ID `4663` before signing transactions
- Start with a small deposit to test the flow

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Wrong network | Switch wallet to Robinhood Chain (4663) |
| Transaction fails | Check ETH balance for gas |
| Token not visible | Import USDG contract: `0x5fc5360D0400a0Fd4f2af552ADD042D716F1d168` |
| dApp won't connect | Refresh page, reconnect wallet, try another browser |
