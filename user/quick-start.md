# Quick start

Get into HoodPot in a few minutes.

## 1. Get a wallet

Use any EVM wallet that supports custom networks:

- [Robinhood Wallet](https://wallet.robinhood.com/)
- MetaMask, Rabby, or similar

## 2. Add Robinhood Chain

| Field | Value |
|-------|-------|
| Network name | Robinhood Chain |
| Chain ID | `4663` |
| RPC URL | `https://rpc.mainnet.chain.robinhood.com` |
| Currency symbol | ETH |
| Block explorer | `https://robinhoodchain.blockscout.com` |

See [Wallet setup](wallet-setup.md) for step-by-step instructions.

## 3. Get USDG

HoodPot deposits **USDG** — the stablecoin used on Robinhood Chain earn products.

You need USDG in your wallet on chain `4663` before depositing. Check the explorer or Robinhood earn flows for how to obtain USDG on mainnet.

## 4. Open the dApp

Go to **[app.hoodbet.fun](https://app.hoodbet.fun)**.

1. Click **Connect wallet**
2. Approve Robinhood Chain if prompted
3. Open the **Deposit** tab
4. Enter an amount and confirm the transaction

## 5. Stay in the pool

The longer you keep USDG deposited, the more your **time-weighted balance** grows — and the better your draw odds.
